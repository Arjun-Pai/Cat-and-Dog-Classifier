# Cat and Dog Image Classifier

## Overview

This project is a Convolutional Neural Network (CNN) built using TensorFlow and Keras to classify images of cats and dogs. The model is trained using image datasets and uses deep learning techniques to recognize patterns in images.

This project was completed as part of the FreeCodeCamp Machine Learning with Python certification.

## Features

* Image classification for cats and dogs
* Data augmentation to reduce overfitting
* TensorFlow and Keras implementation
* CNN architecture with convolution and pooling layers
* Accuracy visualization during training

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib

## Model Architecture

The model consists of:

* Multiple Conv2D layers
* MaxPooling2D layers
* Dense neural network layers
* Sigmoid activation for binary classification

## Dataset

The project uses a labeled dataset containing:

* Training images of cats and dogs
* Validation images
* Test images for predictions

## How to Run

1. Open the notebook in Google Colab.
2. Run all cells in order.
3. Train the model.
4. Test predictions on unseen images.
5. Run the final FreeCodeCamp test cell.

## Results

The model achieves the minimum required accuracy for the FreeCodeCamp challenge and is capable of predicting whether an image contains a cat or a dog.
